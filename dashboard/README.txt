cara run
+++ btw yang dipake folder dashboard ya +++

1) buka terminal
2) masuk ke direktori /dashboard -> "cd ../direktori"
3) npm install -g @angular/cli
4) ke powershell terus ketik -> Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
5) kembali ke terminal -> ng serve
6) http://localhost:4200/

+++ kalo ngga ada nodeJS, install dulu dan masukin direktori path ke system environment variables bagian PATH +++
